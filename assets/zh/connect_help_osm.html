支持硬件机型

Osmo Mobile 3、DJI OM 4、DJI OM 5、Osmo Mobile SE、Osmo Mobile 6

连接步骤

步骤一
  1. 请确保已激活设备
  2. 请确保开启手机蓝牙（地理位置）
步骤二

方法一：运行畅片 App，首页右上角点击【我的设备】，进入后可搜索设备，搜索成功后，选择连接前缀为 OM 的设备，然后在设备上确认完成连接授权。

方法二：运行畅片 App，开启硬件设备，弹出可连接的设备时，选择连接前缀为 OM 的设备，然后在设备上确认完成连接授权。

方法三：初次连接后，二次连接时（硬件设备正常情况下）可触发自动连接。

连接问题

连接过程中出现“连接超时/连接失败”提示
  1. 请将DJI设备关机重启，并将畅片 App 重启后，尝试再次连接
  2. 重置设备连接状态后，尝试再次连接
无法搜索到设备

  1. 检查设备是否开机

  2. 请确保蓝牙（地理位置）开启

  3. 检查设备是否连接到其他手机

  4. 尝试重置蓝牙连接

    DJI OM 4或Osmo Mobile 3：同时按下「模式切换按键 」+「拍摄按键 」+「M 按键 」并持续 1 秒。重置成功后再次连接Osmo设备。

    Osmo Mobile 6、Osmo Mobile SE或DJI OM 5：同时按下「扳机键 」+「拍摄按键 」+「切换按键 」并持续 1 秒。重置成功后再次连接Osmo设备。
